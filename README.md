## installation

```bash
  # install Node.js (v18.15.0)

  # install yarn (v1.22.19)
  $ npm install --global yarn  

  # clone the projet
  $ git clone https://github.com/Legrand-Malobe/portfolio.git


  # install all dependencies
  $ cd portfolio
  $ yarn
  ```

## Running the app

 ```bash
  # development
  $ yarn run dev
 ```
